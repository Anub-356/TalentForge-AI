# 📜 Changelog

All notable changes to the **TalentForge AI** project will be documented in this file.

## [v3.2.0] - 2026-06-26
### Added
* **Official Redrob Challenge Deliverable:** Finalized production monorepo package.
* **Evidence Verification Engine:** Triangulates candidate claims against external developer telemetry (GitHub repositories, skill assessment scores).
* **6-Rule Honeypot Detector:** Intercepts ATS manipulation, keyword stuffing, and concurrent timeline overlaps.
* **Explainable Evidence Ledgers:** Populates natural language `reasoning` strings for all shortlist rows.
* **Recruiter Command Center:** Glassmorphism React 18 UI ("talentforge-ai-frontend") with zero-state requisition benchmarks.
* **Deterministic Tie-Breaking:** Implemented $O(N \log K)$ min-heap sorting strictly on `-score` and `candidate_id` ascending.
* **Interactive Cloud Sandbox:** Embedded reproducible Colab/Jupyter demonstration notebook (`sandbox_demo.ipynb`).
